//////////////////////////////////////////////////////////////////////////////////////////////// Welcome : Bienvenue sur la documentation du BLEU non officiel, elle te permettra de trouver les astuces dans le jeu et de te guider tout au long de ton aventure. Je te conseil de regarder toutes les sections pour t’informer comment le jeu marche dans son ensemble. Présentation : NationGlory est un PvpFaction moddé. Il se déroule sur la carte du monde. Le but de ce jeu est de devenir la nation la plus forte. Un synthème de guerre entre pays qui ne se fini jamais. Vous pouvez vous faire trahir, former une alliance. La seule chose qui importe c’est la sécurité de votre pays. Les pays sont crées par les joueurs qui construisent des villes pour pouvoir accueillir des personnes dans leur nation. Il y a différents moyens de gagner de l’argent mais celui qui marche le mieux, c’est les notations. Chaque semaine tous les pays seront notés par des membres du staff, plus ton pays est développé en terme de builds, de membres, de guerres alors tu recevras de l’argent en compassion de ton travail. Il y a des commandes qui peuvent se faire depuis l’interface des pays. Attention les factions, c’est les pays donc ça ne change rien. -/f create Cette commande sert à créer un pays. Attention cette commande ne fonctionnera pas si vous êtes dans un pays. Il est possible d'en créer sans cette commande. Tu as juste à appuier sur E pour ouvrir l’inventaire et si tu es sans faction, il y aura écrit "créer ton pays". Tu ne pourras pas créer ton pays s'ils sont tous pris ce que tu peux faire c’est attendre les disbands des pays le mercredi et le samedi à 17h. /f set home Cette commande est exécutable si tu as la permission. Elle permet de créer le sethome du pays. /f f page Permet de visualiser tous les pays. /f f Permet de regarder l’interface d’un pays ciblé. /f f Permet de savoir si un joueur est dans le pays ou s'il est en wilderness. /f stats ou /f s Permet de voir les stats de ton pays. /f war ou /f w Permet de voir tout ce qui est en rapport avec la guerre. /f f Ouvre le panel de pays. /f invite Permet d’inviter un joueur si le pays est fermé au public. /f join Permet de rejoindre un pays. /f leave Permet de quitter un pays. /f kick Cette commande est exécutable si tu as la permission. Elle permet d’exclure un joueur du pays (peut se faire avec le panel nation). /f desc Cette commande est exécutable si tu as la permission. Elle permet de changer la description du pays. /f sethome Permet de définir le point de spawn de la faction. /delhome Permet de supprimer un point spawn. commandes avec permission /f promote joueur Permet de promouvoir un joueur dans le rang au dessus mais il est aussi possible de le faire avec le panel de nation. /f demote joueur Permet de rétrograder un joueur dans un rang en dessous mais il est aussi possible de le faire avec le panel de nation. /f officer joueur Permet de promouvoir un joueur au rang d’officier mais il est aussi possible de le faire avec le panel de nation. /f leader joueur Permet de promouvoir un joueur au rang de Leader mais il est aussi possible de le faire avec le panel de nation. /f ally faction Permet de mettre une faction ennemie en allié. /f truce faction Permet de mettre une faction en confiance. /f neutral faction Permet d’envoyer une demande de neurale a une faction ennemie. /f enemy faction Permet de déclarer la guerre sous autorisation et avec cause. /f claim Permet de mettre un chunk sous ton pays. /f unclaim Permet d’enlever un chunk de ton pays. /f tax Permet de mettre des taxes pour les joueurs du pays. /f salary Permet de mettre des salaires pour les joueurs du pays. ///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////// Panneau de shop Le panneau de shop est un systhème qui permet de faire du commerce grâce à un coffre. C'est un panneau qui permettra, quand le joueur appuiera dessus, d'acheter l’item. Merci de bien lire avant de dire que ça ne marche pas. Exemple de panneaux : ligne 1 : Heatcliff ligne 2 : 40 ligne 3 : B 4$ ligne 4 : ou Iron ou #255 image d’un shop Merci de bien mettre les espaces entre B et le prix. Pour savoir l’id d’un objet prend celui-ci en main et tape "/iinfo", un message ou un numéro apparaîtra, n’oublie pas # avant l’id sur le panneau. Si aucun champ est rempli sur la première ligne. Il sera mis par défaut à ton pseudo. Les couleurs ne marchent pas sur les panneaux de shop. Le panneau fonctionne si l’item vendue est à l’intérieure. Panneaux pour louer Si tu es chef de ton pays, tu peux vendre ou louer des chunks qui appartiennent à ta faction. Mettre le panneau dans le chunk où tu veux louer ou vendre, si tu ne s’est pas où sont tes chunks, appuie sur F9.
