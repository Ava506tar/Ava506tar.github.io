SHOP

Le panneau de shop est un systhème qui permet de faire du commerce grâce à un coffre. C'est un panneau qui permettra, quand le joueur appuiera dessus,d'acheter l’item. Merci de bien lire avant de dire que ça ne marche pas.

Exemple de panneau :

ligne 1 : "Pseudo" Heatcliff
ligne 2 : "Quantité d’item a vendre" 40
ligne 3 : "Mettre le signe B pour dire buy(vendre)" B "Prix" 4$
ligne 4 : "id de l’item" ou "le nom de l’item en anglais" Iron ou #255

Merci de bien mettre les espaces entre B et le prix. Pour savoir l’id d’un objet prend celui-ci en main et tape "/iinfo", un message ou un numéro apparaîtra, n’oublie pas # avant l’id sur le panneau. Si aucun champ est rempli sur la première ligne. Il sera mis par défaut à ton pseudo. Les couleurs ne marchent pas sur les panneaux de shop. Le panneau fonctionne si l’item vendue est à l’intérieure.
