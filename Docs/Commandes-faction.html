Faction

Bonjour, cette partie concerne les commandes factions: pays, colonies, ally, enemy, home. Il y a des commandes qui peuvent se faire depuis l’interface des pays. Attention les factions, c’est les pays donc ça ne change rien.

Crée un pays :

/f create "nom du pays"
Cette commande sert à créer un pays. Attention cette commande ne fonctionnera pas si vous êtes dans un pays. Il est possible d'en créer sans cette commande. Tu as juste à appuier sur E pour ouvrir l’inventaire et si tu es sans faction, il y aura écrit "créer ton pays". Tu ne pourras pas créer ton pays s'ils sont tous pris ce que tu peux faire c’est attendre les disbands des pays le mercredi et le samedi à 14h.

Look un pays :

/f f page
Permet de visualiser tous les pays.

/f f "Pays"
Permet de regarder l’interface d’un pays ciblé.

/f f "Nom du joueur"
Permet de savoir si un joueur est dans le pays ou s'il est en wilderness.

/f stats ou /f s
Permet de voir les stats de ton pays.

/f war ou /f w
Permet de voir tout ce qui est en rapport avec la guerre.

/f f
Ouvre le panel de pays.

Utulitaire :

/f home "nom du joueur"
Permet de se téléporté au point du spawn du pays

/f invite "nom du joueur"
Permet d’inviter un joueur si le pays est fermé au public.

/f join "Pays"
Permet de rejoindre un pays.

/f leave "Pays"
Permet de quitter un pays.

/f kick "nom du joueur"
Cette commande est exécutable si tu as la permission. Elle permet d’exclure un joueur du pays (peut se faire avec le panel nation).

Pays :

/f desc "description"
Cette commande est exécutable si tu as la permission. Elle permet de changer la description du pays.

/f sethome
Cette commande est exécutable si tu as la permission. Elle permet de créer le sethome du pays.

Permission :

/f promote "Pseudo"
Permet de promouvoir un joueur dans le rang au dessus mais il est aussi possible de le faire avec le panel de nation.

/f demote "Pseudo"
Permet de rétrograder un joueur dans un rang en dessous mais il est aussi possible de le faire avec le panel de nation.

/f officer "Pseudo"
Permet de promouvoir un joueur au rang d’officier mais il est aussi possible de le faire avec le panel de nation.

/f leader "Pseudo"
Permet de promouvoir un joueur au rang de Leader mais il est aussi possible de le faire avec le panel de nation.

Ally|Truce|Neutral|Enemy :

/f ally "faction"
Permet de mettre une faction ennemie en allié.

/f truce "faction"
Permet de mettre une faction en confiance.

/f neutral "faction"
Permet d’envoyer une demande de neurale a une faction ennemie.

/f enemy "faction"
Permet de déclarer la guerre sous autorisation et avec cause.

Claim :

/f claim
Permet de mettre un chunk sous ton pays.

/f unclaim
Permet d’enlever un chunk de ton pays.

Tax|Salary :

/f tax
Permet de mettre des taxes pour les joueurs du pays.

/f salary
Permet de mettre des salaires pour les joueurs du pays.
